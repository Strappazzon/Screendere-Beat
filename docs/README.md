## GitHub Pages

The **docs** folder contains an HTML version of [README.md](https://github.com/Strappazzon/Screendere-Beat#readme).

If you are looking for *Screendere Beat* source code, please [see the "src" folder](https://github.com/Strappazzon/Screendere-Beat/tree/master/src).

### Running locally

- Install [Ruby](https://www.ruby-lang.org)
- Run `jekyll s` to serve the website locally
- Navigate to <http://127.0.0.1:4000/Screendere-Beat/>
